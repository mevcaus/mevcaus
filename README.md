# Hey, I'm Mev 👋

CS student at **Boise State University** (3.94 GPA, Dec '26) and **Hack Reactor** alum. I like building things that actually work - mostly on the backend and full-stack side. REST APIs, CI/CD pipelines, databases, that kind of stuff.

Right now I also work at BSU's IT Help Desk, where I help 25,000+ people figure out why their stuff isn't working. I even built an AI support agent with RAG that the team uses more than ServiceNow's own search. So that's cool.

## What I'm working on

- **[JavaDropbox](https://github.com/mevcaus/JavaDropbox)** - A cloud file storage platform built with Java 21, Spring Boot 3, and PostgreSQL. File versioning, ZIP streaming, path traversal prevention, the whole deal.
- **[GoProxy](https://github.com/mevcaus/goproxy)** - A Layer 7 HTTP load balancer written in Go using only the standard library. Round-robin and least-connections routing, health checks, lock-free concurrency with `sync/atomic`.
- **[Stackage Server](https://github.com/StackageApp/stackage-server)** - Node.js/Express backend for a social mobile app with real-time sync via Cloud Firestore.
